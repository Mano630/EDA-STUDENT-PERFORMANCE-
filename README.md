# EDA-STUDENT-PERFORMANCE
# Student Performance Analysis (EDA)

## Introduction

This project performs **Exploratory Data Analysis (EDA)** on the *Student Performance in Exams* dataset to understand the factors influencing students' academic performance. The analysis investigates how demographic, social, and educational attributes affect scores in Mathematics, Reading, and Writing.

## Objective

The primary objectives of this project are:

* Analyze student performance across different subjects.
* Identify relationships between test scores and demographic factors.
* Examine the impact of parental education, lunch type, and test preparation courses.
* Discover trends, patterns, and insights that can support educational decision-making.
* Prepare the dataset for future machine learning model development.

## Key Features

* Comprehensive data exploration and cleaning.
* Missing value and duplicate value analysis.
* Statistical summary of numerical features.
* Categorical feature analysis.
* Score distribution visualization.
* Gender-wise performance comparison.
* Impact analysis of parental education levels.
* Test preparation course effectiveness evaluation.
* Correlation analysis among academic scores.
* Insight generation through data visualization using Matplotlib and Seaborn.

## Dataset Information

* **Dataset Name:** Student Performance in Exams
* **Source:** Kaggle
* **Rows:** 1,000
* **Columns:** 8

### Features

* Gender
* Race/Ethnicity
* Parental Level of Education
* Lunch
* Test Preparation Course
* Math Score
* Reading Score
* Writing Score

* Student performance varies across demographic and educational factors.
* Test preparation courses generally improve exam scores.
* Higher parental education levels tend to correlate with better academic performance.
* Reading and writing scores exhibit a strong positive relationship.
